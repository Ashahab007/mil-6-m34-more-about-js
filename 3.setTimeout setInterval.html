Event Loop

যখন call stack খালি হয়ে যায় তখন event queue থেকে Asynchronous task টা call stack এ পাঠায় দেয় একটা একটা করে। একটা Asynchronous task call stack complete করার পর তা যখন call stack থেকে খালি হয়ে যায় তখন আরেকটা Asynchronous task call stack এ পাঠায় দেওয়া হয়। এই occurance টাকেই event loop বলে।
